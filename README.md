# Spotify Hit Song Prediction

> Machine Learning project for predicting whether a Spotify track becomes a hit song using audio features and statistical learning methods.

---

# Overview

This project aims to predict whether a song will become a hit based on 15 different Spotify audio features such as Danceability, Energy ect..
Using classification models, we analyze patterns behind successful songs and compare different machine learning approaches to identify the best-performing model.

---

# Objective

The goal of this project is to:

- Analyze Spotify audio feature data
- Build predictive classification models
- Compare model performance
- Identify important features related to hit songs
